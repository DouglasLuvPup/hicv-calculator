# HICV Reservation Calculator

Holiday Inn Club Vacations (HICV) Reservation Windows Calculator

## Features
- Calculate 1st Day of Reservation Window from today's date
- Calculate Earliest Booking Date from desired travel date
- View all reservation windows at a glance

## Usage
Visit: https://DouglasLuvPup.github.io/hicv-calculator

Works on any device - PC, iOS, Android!
